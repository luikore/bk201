= bk201

In-memory binary code runner, so you can use very fast asm in ruby.

currently Win XP and Ruby1.9+ only (due to runnable memory alloc issues).

an asm DSL is under development.

= requirement

32-bit x86 machine (still learning 64-bit machine code ..).

a C compiler when installing (not required when running).
